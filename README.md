# Python Employee & Project Data Management

## Overview
A Python-based data analysis project completed as part of the Skillovilla Data Analytics certification. The project involves cleaning, transforming, and merging three real-world-style datasets using Pandas and NumPy across 10 structured tasks.

## Datasets
- **Employee** — 5 employees with ID, name, gender, city, age
- **Project** — 14 projects with cost, status (Finished/Ongoing/Failed)
- **Seniority Level** — designation levels per employee

## Tasks Covered
1. Created DataFrames and exported as CSV files
2. Imputed missing project costs using a running-average For loop
3. Split name column into First Name and Last Name
4. Merged all 3 datasets into a single Final DataFrame
5. Added 5% bonus for employees with Finished projects
6. Demoted designation level for Failed projects; removed ineligible employees
7. Added Mr./Mrs. prefix and dropped gender column
8. Promoted designation for employees aged over 29
9. Computed total project cost per employee
10. Filtered employees by city name containing the letter "o"

## Tools Used
- Python, Pandas, NumPy

## Key Skills Demonstrated
- Data cleaning and transformation
- Handling missing values
- DataFrame merging and joining
- Conditional logic and filtering
- Iterative computation using loops
