# Python-Employee-Project-Analysis
Data wrangling and transformation project using Python (Pandas &amp; NumPy) — cleaning, merging, and analyzing employee and project datasets across 10 structured tasks.

Python Employee & Project Data Management
Overview
A Python-based data analysis project completed as part of the Skillovilla Data Analytics certification. The project involves cleaning, transforming, and merging three real-world-style datasets using Pandas and NumPy across 10 structured tasks.
Datasets

Employee — 5 employees with ID, name, gender, city, age
Project — 14 projects with cost, status (Finished/Ongoing/Failed)
Seniority Level — designation levels per employee

Tasks Covered

Created DataFrames and exported as CSV files
Imputed missing project costs using a running-average For loop
Split name column into First Name and Last Name
Merged all 3 datasets into a single Final DataFrame
Added 5% bonus for employees with Finished projects
Demoted designation level for Failed projects; removed ineligible employees
Added Mr./Mrs. prefix and dropped gender column
Promoted designation for employees aged over 29
Computed total project cost per employee
Filtered employees by city name containing the letter "o"

Tools Used

Python, Pandas, NumPy

Key Skills Demonstrated

Data cleaning and transformation
Handling missing values
DataFrame merging and joining
Conditional logic and filtering
Iterative computation using loops
